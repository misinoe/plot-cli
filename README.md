# plot-cli

「物語の体操」	大塚英志 (著)

第一講、6枚のカードを引き物語を作るレッスンをCLIで行えるようにした物です。


## Install

```bash
$ npm install -g misinoe/plot-cli
```


## Usage

```bash
$ plot gen
```

## Example

```json
{
    "1.主人公の現在": "治癒",
    "2.主人公の近い未来": "<逆>調和",
    "3.主人公の過去": "善良",
    "4.援助者": "<逆>知恵",
    "5.敵対者": "<逆>慈愛",
    "6.結末(目的)": "<逆>節度"
}
```
